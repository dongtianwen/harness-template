## 项目初始化（手动准备）
**在开始前，请务必执行以下手动操作：**

1. **运行启动脚本**
   双击运行 `scripts\start.bat`，它会自动为你安装 pre-commit 钩子并拉取最新代码。每次开始工作前都推荐运行一次以获取项目状态。

2. **配置 AI 维护密钥（必须）**
   后台维护脚本使用 NVIDIA NIM 进行自我报告。在系统环境变量中添加你的 API 密钥：
   `变量名：NVIDIA_API_KEY`
   `变量值：你的 NVIDIA API 密钥`

3. **设置开机自动维护（可选）**
   如果你希望电脑每天开机自动跑一次状态检查：
   打开文件资源管理器，进入 `scripts\` 文件夹，**右键点击** `setup_maintenance_scheduler.bat` -> **以管理员身份运行**。

## 快速开始
1. 查看 `docs/overview.md` 了解背景
2. 查看 `STATE.md` 了解当前进度
3. 查看 `tasks/` 认领或新建任务
4. 执行完毕后更新 `reports/`

## 目录说明
| 目录/文件 | 用途 |
|-----------|------|
| PROMPTS.md | 触发语手册，包含各步骤如何唤醒与指引 AI 工作的指令，**使用者必读** |
| AGENTS.md | AI Agent 行为规范与强制规则（AI 会在每次任务前自动读取执行原则）|
| PRD.md | 产品需求文档，由架构设计工作流生成，AI 工作时的技术标准输入 |
| docs/ | 所有上下文文档（AI 读这里获取信息）|
| tasks/ | 任务定义（AI 从这里接任务）|
| reports/ | 执行报告（AI 往这里写结果）|
| workspace/ | 临时工作区、草稿、实验产物 |
| src/ | 正式源代码 |
| tests/ | 测试代码 |
| scripts/ | 脚本工具 |
| logs/ | 运行日志 |
